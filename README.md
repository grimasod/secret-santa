# secret-santa

## About
This is a demo app, with no practical use.

Technology Stack:
- Nuxt.jx
- Vue.js
- Vuex
- Tailwind CSS
- API requests (randomuser.me)

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start
```
